CSS

Git

  • Add notes and different commmands about Git here.
  • git status: checks what branch we are currently on.
  • ls : views all files in the selected directory.
  • git checkout -b branch-name: this creates a new branch and switches to it.(used for committing changes to your main branch).
  • cd ..: changes the directory by putting you in the main folder or directory.
  • git add -A: add all changes to the staging area.
  • git commit -m: committing the changes to our repository. m- is adding a message for the changes.
  • git pull origin main: pulles the base branch into the featured branch to sync them before pushing code.
  • git push origin (enter branch here):pushes the branch to github.

JavaScript

  • To link the JavaScript file to the html, you link the script under the footer of the text file.
  • JavaScript has built in math properties check it out: Here.
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • A function is a block of code that is designed to perform a task and is executed when called, and makes code more reuseable.
  • An array in JavaScript is a type of global object that is used to store data. Arrays consist of an ordered collection or list containing zero or more data types, and use numbered indices starting from 0 to access specific items.
  • An object is a collection of different properties, and is associated with a name and a value.
